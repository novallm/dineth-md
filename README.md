---

### **Quick Start**
1. Clone repository  
   ```bash
   git clone https://github.com/novallm/dineth-md.git
   ```

2. Install dependencies  
   ```bash
   cd dinethmd-bot
   npm install
   ```

3. Start the bot  
   ```bash
   npm start
   ```

4. Scan the QR code with WhatsApp.
