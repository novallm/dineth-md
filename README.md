## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- MongoDB
- FFmpeg

### Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/dinethmd-bot.git
```

2. Install dependencies
```bash
cd dinethmd-bot
npm install
```

3. Configure environment
```bash
cp .env.example .env
# Edit .env with your credentials
```

4. Start the bot
```bash
npm start
```
